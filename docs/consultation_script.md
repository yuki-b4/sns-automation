# 個別相談スクリプト（45分・無料）

最終更新: 2026-05-12

> 本ドキュメントはセミナー出席者向け個別相談（45分・無料／通常¥50,000/h相当 ＝ 45分換算 ¥37,500相当の希少性）のオペレーション・スクリプト。バックエンド「愛を深め続けるマインド構築講座（¥550,000）」へのクロージングを目的とする。
>
> ファネル全体設計は `docs/funnel_design.md` を参照。本スクリプトは §2.5 個別相談会の運用版。

---

## 1. 設計原則（11原則）

### 1.1 押し売りしない、ただし「変わる」決断は支援する
強引なクロージングはしない。同時に、「半分以上やりたい」と思っている人の背中は押す（Principle 11）。判断基準は §3.4 ステップ3 と §3.7 で示す。

### 1.2 「学ぶ」ではなく「変わる」フレームを冒頭で提示
ペルソナは知識学習癖あり（心理学／脳科学／コーチング経験あり）。「もっと学べば解決する」と思いがちな層なので、個別相談は**「学ぶ場」ではなく「変わる選択を見つける場」**と冒頭で明示する。

### 1.3 言語化セッションそのものを無条件の価値として提供する
通常¥50,000/h相当の中身。契約有無に関わらず参加者は「自分の前兆が分かった」を持ち帰る。これが押し売りせずに成約率50%を達成する設計の核。

### 1.4 観察者ポジションを保つ
クライアント事例は「クライアントの〇〇さん」形式で語り、自分の体験と区別（`docs/author_profile.md` 準拠）。結果保証や成功事例の直接引用は避ける。

### 1.5 「ワクワクする目標」を本人の口で言語化させる
ベネフィットへの臨場感を作る。ヒアリングで未来軸（3年後・5年後）を聴き、本人に「100%やりたい」状態を引き出す。これが後段の天国/地獄パートの素材になる。

### 1.6 天国と地獄を両方見せる
- **天国**：講座を受けて変わった未来（短期/中期/長期）
- **地獄**：今のパターンを続けた場合の3〜5年後の現実

両方の臨場感を出して「理想と現実のギャップ」を体感させる。脅しではなく事実として淡々と語る。

### 1.7 理想と現実のギャップを気づかせる
ヒアリングで聴いた理想（質問1の回答）と、過去5年のパターンを続けた場合の未来を並置し、ギャップの大きさを本人に認識させる。「今までの考えと行動の結果が今の貴方。今すぐそれができるか」を考えてもらう。

### 1.8 自己流（事故流）リスクを提示する
自分一人で言語化を続けるのと、現在進行形で関係を深め続けている人と一緒に取り組むのでは、定着までの時間が全く違う。ペルソナの pain #3／#4（知識はあるが応用できない／テクニックでは解けない）を解く構造として示す。自己流は事故流。

### 1.9 「今、解決が必要」な課題を明確にする
「次の関係に入る前に内側設計を整えるのが最も効率」「相手ができてからでは難易度が上がる」を伝え、現在性を提示する。ただしFOMO演出はしない。

### 1.10 自己投資の必要性はセッション前に仕込む
本スクリプトは、自己投資の必要性が **セッション前（LINE配信／セミナー）で既に伝達済み** である前提で組まれている。LINE Day 4／Day 5／セミナー前半でこの前提を作っておくこと（§4.3 参照）。

### 1.11 「半分以上やろう」が決断ライン
即決させず、検討期間48時間を提示。クロージング時の決断ラインは**「半分以上やってみたいと思えているか」**。背中押しのオプション：分割払い／本日割／クーリングオフ／プレゼント特典（運用者設定次第）。

### テーマ一貫
「親密になるほど窮屈になっていた関係を、関係を壊しそうな前兆に気づくことで、愛が深まるほど安心できる関係に変える ── 結婚の幸せに踏み込める」を軸に進める。

## 3. 各セクションのスクリプト

### 3.1 オープニング（0-5分）

**話す内容**:

> 〇〇さん、本日は45分のお時間をいただきありがとうございます。
>
> 最初に1つだけお伝えします。今日は「学ぶ場」ではなく、〇〇さんが「変わる選択」を見つける場にしたいと思っています。
>
> 心理学や脳科学の知識は、本やコンテンツで十分手に入る時代です。でも、〇〇さんがここに申し込んでくださったのは、知識を増やしても自分の問題が解決しなかった、という感覚があるからだと思います。
>
> 今日のゴールは2つあります。
> 1つ目は、〇〇さんが過去の恋愛で『親密になるほど窮屈になる』感覚を持っていた瞬間を、一緒に言語化すること。
> 2つ目は、〇〇さんが「3年後・5年後にどんな関係性の中にいたいか」を改めて言葉にしていただき、そこに対する〇〇さん専用のマインド設計の方向性を見つけることです。
>
> 先日のセミナーで『愛を深め続ける関係性構築戦略』をお伝えしましたが、今日はそれを〇〇さんの具体的なケースに落とし込んでいきます。
>
> 進行は、最初の10分で過去を、次の5分で未来を聴かせてください。そのあと10分で一緒に前兆を言語化し、5分で現状と理想のギャップをお見せします。最後の10分で講座のご紹介と質問の時間を取ります。
>
> なお、この45分間は通常¥50,000/h相当のコーチング枠です。今回はセミナーにご参加いただいた方限定で無料でお受けしていますが、それは「無料の情報は無料」という意味ではなく、「セミナーで真剣に時間を投資された方への、変わるための入口」として設計しています。
>
> それでは始めましょう。

**チェックリスト**:
- [ ] 「学ぶ→変わる」フレーム提示（原則1.2）
- [ ] ゴール2点（過去言語化 ＋ 未来設計）を明示（原則1.5への布石）
- [ ] 進行構造を予告
- [ ] ¥50,000/h相当の希少性 ＋「無料は無料の意味じゃない」のメタ提示（原則1.10再確認）
- [ ] 質問タイムを最後に集約

### 3.2 現状ヒアリング（5-15分・10分）

3軸で過去を聴く。時間配分は厳守。

#### 軸1: 恋愛経験のリスト化（2分）

**質問**:
- 「これまでの恋愛経験、ざっくり何回くらいありますか？」
- 「その中で、最も印象に残っている関係を1つ教えていただけますか？」

#### 軸2: 親密になるほど窮屈になった瞬間（3分）

**質問**:
- 「その関係の中で、〇〇さんが『窮屈だな』『冷めるな』『重いな』と感じた瞬間を覚えていますか？」
- 「それは関係のどの段階で起きましたか？（出会って数ヶ月／親密期／結婚を意識した瞬間 等）」
- 「その時、〇〇さんは何をしましたか？（距離を取った／別れた／我慢した／話し合った）」
- 「相手から最後に言われた言葉で、印象に残っているものはありますか？」

#### 軸3: 関係を壊しそうな前兆の体感（5分）

**質問**:
- 「振り返って、〇〇さんが関係を壊しそうになる『前兆』を3つ挙げるとしたら何ですか？」
- 「それは、思考？感情？行動？どれが先に来ますか？」
- 「その前兆が出た時、〇〇さん自身は何が起きていると思いますか？」

**ヒアリング中の内的チェック（口に出さない）**:
- [ ] pain #1（別れ際の言葉のトラウマ）
- [ ] pain #2（自責⇔他責ループ）
- [ ] pain #5（結婚への踏み込み難）
- [ ] pain #7（一人で抱える孤独）
- [ ] 反復パターンの自覚

**ヒアリング作法**:
- 答えにくそうな質問は深追いしない
- 「正解」を求めず、その人の体感を聴く
- 沈黙を恐れず、思考の時間を与える
- 共感の相槌は入れるが、解釈は急がない

### 3.3 理想・目標ヒアリング（15-20分・5分）

ワクワクする未来像を引き出し、ベネフィットへの臨場感を作る（原則1.5）。後段の天国/地獄パートで使う素材を作る。

**質問**（順番に投げる）:

- **質問1（1分）**：「ここで一度、未来の話に切り替えたいです。〇〇さんは3年後・5年後、どんな関係性の中にいたいですか？できれば具体的に、朝起きてから夜寝るまでの1日を、相手と一緒にどう過ごしているか想像してみてください」
- **質問2（1分）**：「その理想の自分は、今の自分と比べて何が違いますか？マインド面で。表情で。話し方で」
- **質問3（1分）**：「もし結婚相手と一緒にその日々を送れているとしたら、〇〇さんの仕事はどう変わっていそうですか？」
- **質問4（1分）**：「その理想に対して、今、100%『そうなりたい・なれる』と思えますか？それとも『無理だろうな』とどこかで諦めていますか？」
- **質問5（1分）**：「もし諦めかけているなら、何が一番のブロックになっていますか？」

**狙い**:
- 「ワクワクする目標」を本人の口で言語化させる（臨場感の自家発電）
- 100%やりたい状態に到達しているか、どこにブロックがあるかを把握
- 後段の天国/地獄パートで「この理想 vs 続けた場合の現実」のコントラストを使う素材を作る

**注意事項**:
- 「無理」「諦めている」を否定しない。それも事実として受け止める
- 「100%やりたい」と言えなくても問題視せず、講座提案時の判断材料にする
- ワクワクが出てこない場合、過去ヒアリング軸2の「失った理想」を発掘する

### 3.4 言語化セッション（20-30分・10分）

「自分の前兆パターン」を一緒に文字にする。「学ぶ→変わる」の意識転換を含める（原則1.2）。

#### ステップ1: 共通構造の提示（3分）

**話し方の型**:

> お話を聴いて、〇〇さんの場合は『[共通構造の名前]』というパターンが見えてきました。これはセミナーでお伝えした[前兆A型／B型／C型]に近いです。
>
> 具体的には:
> - 〇〇さんは [きっかけ：相手が踏み込んでくる／結婚の話題 等] が起きると
> - [思考：「この人で本当にいいのか」など] が浮かび
> - [感情：急な冷め／窒息感] が出てきて
> - [行動：距離を取る／本音を隠す] に出る
>
> このループの中で、最初に出てくる『前兆』は [X] だと思いますが、いかがでしょうか？

#### ステップ2: 共同言語化（5分）

ペルソナと一緒に書き出す:
- 前兆の正体（一言で）
- それが出る具体的なシチュエーション
- 出てきた時に取りうるアクション（過去パターン／新パターン）

**質問**:
- 「この前兆を言葉にすると、〇〇さん自身の中で何が変わりそうですか？」
- 「これに気づけたら、次の関係でどうしたいですか？」

#### ステップ3: 「学ぶ→変わる」の意識転換 ＋ 確認（2分）

**話す内容**（原則1.2）:

> ここまでで〇〇さんは、ご自分のパターンを「知識として」かなり深く理解されています。でも、〇〇さん自身もお気づきだと思いますが、知識として知っていることと、実際の関係性の中で「あ、今これが起きている」とリアルタイムで気づけることは、まったく別の能力です。
>
> 心理学の本を100冊読んでも、自分が窮屈になった瞬間に「これは私の[パターン名]だ」と気づける人はほとんどいません。これは「学ぶ」では到達できない領域で、「変わる」プロセスが必要になります。
>
> 〇〇さんが先ほど挙げてくださった理想 [質問1の回答を引用] を実現するには、知識を増やすのではなく、自分の内側に気づき続けられる状態を作ることが必要です。

**核心の問い**:

> いま、〇〇さんの中で『自分の前兆パターンが分かった』感覚はありますか？

**応答パターンと対応**:

| 応答 | 解釈 | 次のアクション |
|---|---|---|
| 「はい、分かった／腹落ちした」 | 講座提案準備OK | 天国/地獄パートへ進む |
| 「まだ分からない」 | 言語化不足 | ステップ2に戻り、別角度で試みる |
| 「分からないけど、何かが見えた」 | グラデーション | 天国/地獄パートで確認しつつ進める／講座提案は弱めに |
| 「これは自分のことじゃない」 | パターン認識ミス | ヒアリングからやり直す or 講座提案せず別経路（ミドルエンドnote等）を案内 |

### 3.5 天国・地獄 ＋ 理想と現実のギャップ提示（30-35分・5分）

#### ステップ1: 地獄を見せる（2分）

**話す内容**（原則1.6・1.7・1.9）:

> 〇〇さん、ここで少し残酷な話をします。
>
> 先ほど聴いた[過去のパターン]は、〇〇さんが過去〇年間、複数の関係性で繰り返してきたものです。これは偶然ではなく、〇〇さんの内側の構造から生まれているパターンです。
>
> もし、このまま何もせずに次の関係に入ったとして、3年後・5年後、〇〇さんは [質問1の理想] のような状態に到達できていると思いますか？
>
> 私がこれまでクライアントの方々を観察してきた中で、同じパターンを繰り返す方の3〜5年後は、大きく2つに分かれます。
> 1つ目は「諦める」：結婚への期待を下げ、独身であることや、深まらない関係に慣れて生きる
> 2つ目は「同じことを繰り返す」：出会いと別れを反復し、その都度「タイミングが合わなかった」と自分を納得させ続ける
>
> どちらも、〇〇さんが本当に望んでいる未来ではないと思います。

**注意**:
- 脅しではなく、事実として淡々と語る
- 観察者ポジションを保つ（「クライアントの方々を観察してきた」）
- 「諦めるか、繰り返すか」の2択は強い表現なので、声色は穏やかに

#### ステップ2: 天国を見せる（2分）

**話す内容**（原則1.4・1.6）:

> 一方で、ここで内側に向き合うことを選んだクライアントの方々は、3〜5年後にこう変わっています。
>
> - **短期（最初の1ヶ月）**：[前兆X] が出た瞬間に「あ、これは自分のパターンだ」と気づける状態
> - **中期（3ヶ月）**：気づいた上で、過去とは違う選択を取れる
> - **長期（6ヶ月〜1年）**：関係を深め続けるマインドが土台として整い、相手と一緒に安心して結婚の幸せに踏み込めるようになる
>
> 〇〇さんがおっしゃっていた[質問1の理想]に、確実に近づいていきます。
> これは「学ぶ」では到達できない領域で、「変わる」体験が必要です。

#### ステップ3: ギャップを共有（1分）

**話す内容**（原則1.7・1.9）:

> 今日のヒアリングで明らかになったのは、〇〇さんの理想 [質問1] と、今のパターンを続けた場合の未来の間に、大きなギャップがあるということです。
>
> このギャップを埋めるには、今までと同じことをしていてもダメで、何か新しい取り組みが必要です。
>
> しかも、これは「次の関係が始まる前」が最も効率の良いタイミングです。関係が始まってから内側を変えるのは、相手の存在が変数として絡んでくる分、難易度が上がります。
>
> それを〇〇さんがどう選ぶかは、〇〇さんの自由です。

### 3.6 講座概要・自己流NGの根拠（35-40分・5分）

#### ステップ1: メリット・ベネフィット明示（3分）

**話す内容**（原則1.4）:

> 〇〇さんの場合、このギャップを埋めるために最適な道筋として、私が運営している『愛を深め続けるマインド構築講座』をご紹介します。
>
> [講座概要]
> - 期間: 〇ヶ月（運用者が確定）
> - 価格: ¥550,000
> - 内容:
>   - **個別のマインド設計セッション**（隔週）── 〇〇さん専用の前兆設計を継続的にアップデート
>   - **関係性構築戦略の実践フィードバック** ── 実際の出会い／関係性の中で前兆に気づけたかを毎週レビュー
>   - **困った時のLINE個別サポート** ── 関係性で揺れた瞬間にリアルタイムで気づける伴走
> - ゴール：〇〇さんが、ご自身の前兆に気づき続けながら、愛を深め続けられるマインドを完成させる
>
> ベネフィットとしては、今日言語化した [前兆X] に気づけるようになることで、次の関係を [質問1の理想] に向けて深めていける土台が完成します。仕事のパフォーマンスにも波及するので、結果として人生全体が整います。

#### ステップ2: 自己流（事故流）NGの根拠（2分）

**話す内容**（原則1.8）:

> ここで1つ伝えておきたいのが、「自分一人で取り組む」選択についてです。
>
> 〇〇さんは今日言語化されたものを、ご自分で続けることもできます。実際、本を読み続けたり、毎週ジャーナリングしたり、瞑想したりして取り組まれている方もいます。
>
> ただ、私がこれまで観察してきた限り、自己流で続けている方の大半は、3ヶ月以内に元のパターンに戻ります。理由は単純で、自分が窮屈になった瞬間にそれが「いつものパターンだ」と気づくのは、外からのフィードバックがあって初めて可能になるからです。**自己流は事故流になりやすい**、というのが現場の実感です。
>
> 〇〇さんが知識として既に持っている心理学・脳科学は、自分の問題には応用しにくい設計になっています。応用できるようにするには、現在進行形で関係を深め続けている人と一緒に取り組むのが、最も時間効率が良い方法です。
>
> 私自身も、自分の関係性パターンと向き合う過程で、コーチングを受けながら進めました。一人で本を読んでいた時期と、コーチと取り組んだ時期とでは、変化の速度がまったく違いました。

### 3.7 背中押し・検討期間提示・Q&A（40-45分・5分）

#### ステップ1: 「半分以上やりたい」決断ラインの提示（2分）

**話す内容**（原則1.1・1.11）:

> ここまで聴いていただいてありがとうございます。〇〇さんに2つだけお伝えしたいことがあります。
>
> 1つ目：**「半分以上『やってみたい』と思えていたら、踏み込んでみてください」**。これが私が普段クライアントの方にお伝えしている、決断の目安です。100%確信できる選択肢は人生にほぼ存在しません。半分以上やりたいなら、やってみる価値があると思います。
>
> 2つ目：**「半分未満しか『やりたい』と思えないなら、今は決めない」**も立派な選択です。続かないことを始めても、お互いにとって時間の無駄になります。
>
> いま〇〇さんの中で、「やってみたい」気持ちはどのくらいですか？数字で表すとしたら何割くらいでしょうか？

**応答パターン**:

| 応答 | 対応 |
|---|---|
| 「7割／8割／それ以上」 | 次の検討期間提示へ進む。背中押しオプション提示 |
| 「5割／6割」 | 「半分以上ですね。何が引っかかっていますか？」と聴く → 解消できる懸念なら対応、価格や時期の問題なら検討期間提示へ |
| 「3割／4割」 | 「今は決めないことをお勧めします。今日得たものは持ち帰ってください。3〜6ヶ月後にまたお会いする機会があれば、その時に判断してください」 |
| 「分からない」 | 一度数字を聴く（「強いて言えば何割？」）。答えられない場合は3割未満として扱う |

#### ステップ2: 検討期間 ＋ 背中押しオプション提示（1分）

**話す内容**（原則1.11）:

> 即決していただく必要はありません。検討期間として48時間をお取りします。
>
> [運用者が事前に設定した背中押しオプションを提示。設定がない項目は提示しない]
>
> - **分割払い**：一括／分割（〇回まで）どちらも選べます ※運用者の決済設定次第
> - **本日割**：今日中にご決断いただいた場合、〇〇円割引します ※運用者設定次第・FOMO演出に陥らないよう注意
> - **クーリングオフ**：開始後〇日以内なら全額返金 ※運用者規約次第
> - **プレゼント特典**：ご決断いただいた方には [補助教材／追加セッション] をお付けします ※運用者設定次第
>
> 48時間以内にご返答がない場合は、今回のご案内は一旦クローズとさせていただきます。それで何の問題もありません。〇〇さんが今日の45分で得たものは、講座を取らなくても活きるはずです。

**運用注**:
- 背中押しオプションは運用者が事前に設定しておく。何も設定しない場合は提示しない（押し売り感の回避）
- 「本日割」は FOMO 的に使うと押し売りに転びやすい。原則的には推奨せず、使う場合は理由を明示（例: 「決断疲れを避けるため期間を区切る目的」）

#### ステップ3: Q&A（2分）

> 何か、この時点でご質問はありますか？

質問内容に応じて §4 質疑応答対応を参照。

## 4. 質疑応答対応

### 4.1 よくある質問と回答方針

| 質問 | 回答方針 |
|---|---|
| 「価格が高い」 | 「¥550Kは6ヶ月のマインド構築投資。〇〇さんの理想 [質問1の引用] を実現するためのコストと、その理想を諦め続ける時間的・精神的コストを比較してみてください」 |
| 「自分にできるか不安」 | 「クライアントの皆さん同じ感覚から始まる。できる/できないを決めるのは才能ではなく、伴走の有無」（原則1.8 を再強調・「できない」と思わせない） |
| 「すぐに結果が出ますか」 | 「マインドは時間がかかる。3ヶ月後から手応えが出る人が多い。即効性を求めるなら別の選択肢の方が向いている」 |
| 「自分一人で頑張れる気もする」 | §3.6 ステップ2 を再強調。「自己流は事故流になりやすい」観察データを再提示（原則1.8） |
| 「もう少し考えたい」 | 「48時間お取りします。LINEで質問可。期間は決断疲れを避けるため設定」 |
| 「次の交際相手ができてから受けたい」 | 「相手がいない時のほうが内側設計に集中できる。関係が始まってから内側を変えるのは難易度が上がる」（§3.5 ステップ3 再強調・原則1.9） |
| 「分割払いはできますか」 | 運用者の決済設定次第（事前確認） |
| 「返金保証はありますか」 | 運用者規約次第（事前確認） |
| 「半分はやりたいけど半分迷っている」 | 「半分以上やりたい気持ちがあるなら踏み込んでみてください」（原則1.11 再適用） |
| 「他のサービスと比較中」 | 比較対象の特徴を1度だけ説明。具体的な他社批判はしない |

### 4.2 禁止フレーズ

- 「今だけ」「特別価格」（焦らせる表現／原則1.1 違反）
- 「絶対」「必ず」「100%」（結果保証・原則1.4 違反）
- 「他の人は皆」（同調圧力）
- 「あなただけ特別」（特別感の演出）
- 「これを逃すと次はない」（FOMO 演出）
- 「クライアントの〇〇さんも[具体的成功談]」を結果保証として使う（原則1.4 観察者ポジション逸脱）

### 4.3 セッション前に仕込んでおくべき事項（LINE／セミナー側設計）

本スクリプトは、以下が **セッション前に既に伝達済み** である前提で組まれている（原則1.10）。LINE配信／セミナーで以下を仕込むこと:

| 仕込み事項 | 場所 |
|---|---|
| 自己投資の必要性（本物の情報は無料では手に入らない） | LINE Day 4「テクニック型では解けない理由」／セミナー前半 |
| 「学ぶ」では届かない領域があること | LINE Day 4・Day 5 ／ セミナー前半 |
| 発信者自身が自己流で躓いた経験 | LINE Day 5「発信者の脱却プロセス」／ セミナー60-75分 |
| 「現在進行形で関係を深め続けている人と取り組む」価値 | セミナー全体で発信者がそのポジションを示す |

これらが弱いと、本スクリプトの §3.6 ステップ2（自己流NGの根拠）で初めて聞く話になり、説得力が下がる。セッション前段で繰り返し触れることで、相談時には**再確認のみ**で済む状態を作る。

## 5. 検討期間中のLINE対応

参加者から検討期間中に質問がきた場合の対応方針:

| 質問タイプ | 対応 |
|---|---|
| 内容の追加質問 | 1回まで詳細に回答。2回目以降は「個別に話した方が早いので30分の追加相談を設けますか？」と提案 |
| 価格の交渉 | 「価格はファネル全体の設計上、変動できません」と一貫して返す |
| 期間延長依頼 | 24時間まで延長可・それ以上は再案内サイクル（次のセミナー後の相談）に回す |
| 申込意思表示 | 決済リンクと契約書を即送付 |
| 「他のサービスと比較中」 | 比較対象の特徴を1度だけ説明。具体的な他社批判はしない |

### 48時間経過時の処理

- **申込なし・連絡なし**: 自動クローズ。追加メッセージは送らない（押し売り回避・原則1.1）
- 「次回機会があれば」と来た: 「次の機会はLINE配信でお知らせします」と返し、`status='cooling'` で `ファネル登録ログ` に記録

## 6. 失注時の対応

講座契約に至らなかった参加者への対応:

1. **離脱させない**: LINEブロックされない範囲で配信継続
2. **再アプローチのタイミング**: 3〜6ヶ月後、新しいセミナー案内 or ミドルエンドnote（「理想の相手の見つけ方ガイド」シリーズ）案内
3. **記録**: `ファネル登録ログ` の `status` を以下のいずれかに更新
   - `lost`: 明確に「受けない」と回答した
   - `cooling`: 「いつか機会があれば」など曖昧な状態

### 失注理由の分類記録

`ファネル登録ログ` の `notes` 列に以下のラベルで簡潔に記録（後段の改善分析用）:

| ラベル | 意味 |
|---|---|
| `price` | 価格がネックで離脱 |
| `timing` | タイミングが合わない（仕事繁忙等） |
| `partner_needed` | 「相手ができてから」と希望 |
| `unclear_value` | 講座価値が腹落ちしなかった |
| `wants_quick` | 即効性を求めていた |
| `cant_do_it` | 自分にできる気がしないと感じている |
| `self_taught` | 自己流で頑張ろうとしている |
| `other` | その他 |

## 7. ロール別チェックリスト

### 7.1 開始前（24時間前まで）

- [ ] 参加者の `ファネル登録ログ` を確認（セミナー出席日／LINE登録日／申込日時）
- [ ] セミナー後フォームの回答内容を読み込み、ヒアリングの仮説を立てる
- [ ] Zoomリンク／カレンダー招待が参加者に届いているか確認
- [ ] 講座決済リンク（Stripe等）が事前準備されているか確認
- [ ] 当日提示する背中押しオプション（分割／本日割／クーリングオフ／プレゼント）の有無を確認

### 7.2 開始直前（5分前）

- [ ] Zoom起動・参加者の入室確認
- [ ] スクリプト（本ドキュメント）を画面外に開いておく
- [ ] 録画はしない（参加者のリラックスを保つため）

### 7.3 終了直後

- [ ] `ファネル登録ログ` の `consultation_implemented_at` に実施日時を記録
- [ ] `consultation_status` を `done` に更新
- [ ] 判断結果に応じて `status` を更新（contracted / cooling / lost のいずれか）
- [ ] §3.7 ステップ1 の数字（「やりたい度」何割）を `notes` に記録
- [ ] 検討中の場合は48時間タイマーを記録（手動カレンダー）

### 7.4 48時間後

- [ ] 返答なしの場合、自動クローズ（`status='cooling'` または `lost`）
- [ ] 失注理由がはっきりしている場合は §6 のラベルで `notes` に記入

## 8. 11原則のチェックリスト（セッション後の自己振り返り用）

セッション後、運用者自身が以下を◯/△/✗で振り返る:

| # | チェック項目 | 該当セクション |
|---|---|---|
| 1 | 目的やワクワクする目標のヒアリングができたか／ベネフィットに対して相手が臨場感を持てていたか（100%やりたいになっているのが理想） | §3.3 質問1〜5 |
| 2 | 商品が「今必要」と相手に納得させられたか | §3.5 ステップ3 + §3.6 ステップ1 |

| 6 | 理想と現実のギャップに気づかせることはできたか | §3.5 ステップ3 |
| 7 | 自己流（事故流）でできると思わせていないか | §3.6 ステップ2 |
| 8 | 相手が「今、解決が必要」と感じる課題を明確にできたか | §3.5 ステップ3 |

| 10 | 自己投資の必要性はセッション前に伝達済みだったか／本日再確認できたか | LINE/セミナー側 + §3.1 オープニング |
| 11 | 「半分以上やろうと思えていたらやりましょう」と言えたか | §3.7 ステップ1 |
