## 2. テーマ別評価

投稿済み 9 本の `theme_label` を、ペルソナ（30代エンジニア／恋愛・家庭・働き方の交差点）への刺さりという観点で評価する。

### 強テーマ（継続推奨）

**① 「あと5分だけ」が奪う家族の夜（2026-05-01）**
- 強み: 「あと5分」という具体行動と「家族の夜」という喪失対象が一文で接続されており、損失フレームと所属フレームが同居。爬虫類×哺乳類のハイブリッド構造。
- 改善点: 「あと5分」というモチーフは普遍性が高い反面、エンジニア固有性が弱まりやすい。仕事中断コスト／コンテキストスイッチなど、職能語彙との接続で差別化余地。

**② 子どもの話を上の空で聞く罪悪感の正体（2026-05-03）**
- 強み: 「罪悪感の正体」という認知の名付けが提示されており、読者が自己診断的に読み進められる構造。哺乳類脳（所属・承認の毀損）に効く。
- 改善点: 「罪悪感」は再現しやすい語なので、テーマラベル段階で構造名（例: 注意残余／認知負債）と並置すると次回以降のシリーズ化がしやすい。

**③ 優しさが枯渇する瞬間の構造（2026-05-10, target_brain=両方）**
- 強み: 「枯渇」「構造」という語が、感情論ではなく機構的説明を期待させる。両方向（搾取される身体感覚＝爬虫類／関係性での消耗＝哺乳類）を統合できる稀少なテーマ。
- 改善点: 「構造」を約束した以上、本文側で機構図を提示できているかが勝負。テーマラベル単体では訴求は強い。

### 切り口の磨きどころが大きいテーマ

## 3. 採択シグナル分析（運用者キュレーション）

未採択 60 件を眺めると、運用者が**選ばなかった**側には明確な偏りがある。

**(a) 爬虫類脳ストレートの「数字・損失・残機」系が刈り取られている**
- 「30代独身男の選択肢消失曲線／残機を数えてみた」(5/12)
- 「婚期逃しの複利損失／選択肢の複利」(5/25)
- 「同じ別れを繰り返す確率は何％か」(5/21)

- これら定量・残量・確率訴求はすべて未採択。運用者は「数値で煽る型」を意図的に避けている可能性が高い。

**(b) 心理学知識メタ系（学べば学ぶほど効かない）の連続未採択**
- 5/10「学習迷子」/ 5/11「知識武装が招く恋愛麻痺」/ 5/16, 5/23「心理学知識が応用できない」
- 同一クラスタが 4 回提案され全滅。**generate_note.py の提案多様性が落ちているサイン**で、種出しテンプレが特定モチーフに収束している懸念。

**(c) 哺乳類脳の「相談相手がいない孤独」反復**
- 5/10, 5/11, 5/19, 5/20, 5/24 で類似テーマが連投され、いずれも未採択。運用者にとって既出感が強いとみられる。

**(d) 採択された側の共通項**
- 「家族の夜」「罪悪感」「優しさの枯渇」「踏み込みと見極めの停止」など、**具体的場面 × 内側の名付け**の構造。
- 数値訴求や残機メタファーよりも、生活シーンに身体感覚を埋め込む型が選ばれている。

**generate_note.py への含意:**
- 同一週内で同一クラスタ（残機系・心理学知識メタ系・孤独相談系）が3回以上提案されたら冷却するなど、**過去 N 日の theme_label 類似度を提案時に減点する仕組み**が欲しい。
- 爬虫類脳枠を「数値・残機」一辺倒にせず、「身体反応／縄張り反応」型（例: 投稿済「優しさが枯渇する瞬間の構造」系）を優先するヒントが採択履歴から読み取れる。

## 4. Threads → note の一貫性評価

**一貫性は高い。** 投稿済み note のテーマと Threads 投稿のモチーフは強く呼応している:

| Threads モチーフ | 対応する投稿済 note theme_label |
|---|---|
| 「自分が悪い／仕方なかった」往復 [structure] | 該当 posted なし（未採択側「自責と他責の往復」5/15, 5/23 に重なる） |
| 同棲の夜・転職サイトを開く [personal] | 「踏み込みと見極めの停止状態」(5/10) |
| 「お前といると楽」で息がしづらい [dialogue] | 「優しさが枯渇する瞬間の構造」(5/10) |
| 「ずっとこうだといいね」で手が離れる [dialogue] | 「結婚意識で冷める瞬間の正体」(5/9) |
| 日曜の朝「今日どうする？」[dialogue] | 「子どもの話を上の空で聞く罪悪感の正体」(5/3) と地続き |

**ハンドオフ実装上の所感:**
- Threads 側は「身体感覚＋具体シーン＋言い差し」の三点セットで強く問いを立てており、note 冒頭 300 字で回収する材料は十分揃っている。
- ただし「自責／他責の往復」「冷めたと誤認」など Threads で強く問われたモチーフのうち、note 投稿済みに直接対応するものが無いまま、未採択提案にだけ大量に堆積している組み合わせがある（特に [structure][permission] 系）。**Threads で投げた問いの回収率を上げる**ことが、ハンドオフルールの実効性を高める。
- `engagement_design_rules.threads_to_note_handoff` は概念としては機能しているが、**「どの Threads 投稿を回収する note か」をテーマ提案時にメモする欄**が無いと、執筆段階で対応関係が失われやすい。

## 5. 来週の推奨アクション

### 重点的に伸ばすべき切り口

1. **「身体反応として現れる関係性の停止」シリーズ化**
   - 採択実績: 「優しさが枯渇する瞬間の構造」「踏み込みと見極めの停止状態」
   - Threads でも「手が離れる」「身体が固まる」「指が震える」が頻出。身体感覚×関係停止は採択率・一貫性ともに高い。

2. **「具体場面 × 内側の名付け」型**
   - 採択実績: 「子どもの話を上の空で聞く罪悪感の正体」「あと5分だけが奪う家族の夜」
   - 場面（保育園送り／日曜朝／別れ際）を具体化し、その内側で起きている認知に名前をつける型は安定して採択されている。

### 避けるべき／重複気味

- **数値・残機・複利系の爬虫類脳ストレート**（5/12, 5/21, 5/22, 5/25 が連続未採択）。生成側で頻度を絞る。
- **「心理学を学ぶほど効かない」メタ系**（5/10, 5/11, 5/16, 5/23 が全滅）。視点を変えるか、しばらく休ませる。
- **「相談相手がいない孤独」哺乳類脳系**の単純反復（5 回連続未採択）。場面具体性を足さない限り通らない傾向。

- 投稿済みの target_brain がほぼ未記入のため、まず**採択時に target_brain を必ず確定する運用**を入れる。
