## 2. パターン別評価

### 🥇 知的好奇心（★★★★★）
- **強み**: 「忙しいは客観事実ではなく認知の結果」など、定義の書き換えで読者の前提を揺さぶる構造。逆説型×構造分析型の組み合わせが、真面目なビジネスマン層の知的承認欲求にハマっている。
- **改善点**: 生成本数が少なすぎる。3本/4週では検証データが不足。
- **推定**: いいね=高 / 閲覧=高 / コメント=中 / 転換率=中〜高

### ファン化（★★★☆☆）
- **強み**: 場面描写型の冒頭（「退勤の10分前。大型のプルリクをレビュー中で…」）の解像度が高く、ペルソナへの突き刺し力がある。
- **改善点**: 閲覧数30に対していいね2は転換が弱い。場面描写で終わらず、「あなたへの問い」で締めるCTA設計が不足。
- **推定**: いいね=中 / 閲覧=中 / コメント=低 / 転換率=中

### 共感最大化（★★☆☆☆）
- **強み**: 「2年前、私は…」の体験談導入は感情接続に強い。
- **改善点**: 閲覧16.3は全パターン中**最低**。共感止まりで「読み進める動機」が弱い。Before/Afterと失敗談の重複で、訴求の起伏が単調化。**生成過多 × パフォーマンス低**の最悪コンビ。
- **推定**: いいね=低 / 閲覧=低 / コメント=低 / 転換率=低

### 信頼構築（★★★★☆）
- **強み**: 「30名以上のクライアント」「松竹梅3つ」など具体性と選択肢提示が明快。意思決定型読者に強い。
- **改善点**: いいね1.0は低い。共感ステップが薄いため、初見読者にとって距離が遠い。
- **推定**: いいね=低 / 閲覧=中 / コメント=低 / 転換率=高

### 行動変容（★★★☆☆）
- **強み**: ステップ型で実用性が明確。閲覧31は健闘。
- **改善点**: いいね0は深刻。「読んで終わり」になり、感情反応を引き出していない。
- **推定**: いいね=低 / 閲覧=中 / コメント=低 / 転換率=中

## 3. Threads → note の一貫性評価

### テーマ一貫性: ◯（高い）
Threadsの`[structure]`系（「昼休みにちゃんと休んでいる人ほど…」「休日にちゃんと休めていると思っている人ほど…」）は、knowledge_curiosityパターンの逆説型と完全に同じ骨格。`[dialogue]`系（「あと5分だけが30分」「早く帰りたいじゃなくて帰って何する」）はファン化の場面描写型と直結。**Threads資産は十分にnote側に活かせている**。

### 誘導設計: △（部分的）
`[gw_note_promo]`系3本は明確にnote誘導目的だが、Threadsで提示した問いをnote冒頭で**明示的に回収**できているかは不均一。

### threads_to_note_handoff 実践度
- **◯ 達成例**: ファン化記事「30代エンジニアが保育園に向かう前の3分」は、Threads「早く帰りたいじゃなくて帰って何する」をそのまま引用ブロックで冒頭回収。**模範例**。
- **△ 不足例**: 知的好奇心記事「なぜ忙しいという人ほど忙しいままなのか」はThreads「忙しい」系投稿との接続が冒頭でやや弱い。引用ブロックはあるが、Threads原文の再掲ではない。
- **✕ 未達**: 共感最大化記事はThreadsの問いを冒頭300字以内で回収していないものが多い。これが閲覧低迷の一因の可能性。

## 4. 来週の推奨アクション

### 重点活用パターン
**🎯 知的好奇心を最優先で増産**（目標40%へ復帰）
- 理由: いいね11.0/閲覧44と他パターン圧倒。Threads `[structure]`系の在庫も豊富で素材に困らない。
- 来週アクション: 最低3本生成、うち2本投稿。

### 調整が必要なパターン
**⚠️ 共感最大化を抑制**
- 現状: 10本/4週は明らかに過剰生成。
- 改善案: 週次cap=2を生成段階にも適用。共感単独ではなく「共感→構造分析（知的好奇心）」のハイブリッド型に切り替え、Before/After型と失敗談型の重複使用を禁止。

**⚠️ 行動変容のいいね0問題**
- 改善案: ステップ型の前に「あるあるの場面描写」を1ブロック挿入し、感情接続→実用情報の流れに変更。

### 乖離±10%以上の是正
| パターン | 乖離 | 是正アクション |
|---|---|---|
| 知的好奇心 | -28% | 来週生成6本中3本を割当 / Threads `[structure]`を起点に量産 |
| 共感最大化 | +27% | 来週生成は0〜1本に絞る / 既存在庫から投稿 |

### テーマ×パターン組み合わせ案
- **「休日の休み方」× 知的好奇心（逆説型）**: Threads「休日にちゃんと休めていると思っている人ほど月曜の判断速度が落ちている」を冒頭300字で完全回収する記事。
- **「カレンダーの主導権」× 信頼構築（数字型）**: Threads `gw_note_promo`「仕事ができる人のカレンダーは他人の期待で埋まっている」を起点に、定量データで補強。
- **「自分との対話」× ファン化（場面描写型）+ 行動変容（ステップ型）の融合**: いいね0問題の解消検証。
